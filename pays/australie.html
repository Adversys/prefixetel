Comment appeler l’Australie depuis l’étranger

Format général

Pour appeler un numéro australien depuis l’étranger :

Préfixe international + 61 + indicatif régional + numéro local

Exemple fixe (Sydney) :
(02) 9012 3456 → +61 2 9012 3456

Exemple mobile :
04 12 345 678 → +61 4 12 345 678

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 61.
  • Retirez le 0 initial de l’indicatif régional.
  • Ajoutez le reste du numéro.

Exemple depuis la France :
00 61 2 9012 3456

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +61.
  • Retirez le 0 initial.

Exemple :
+61 4 12 345 678

Structure des numéros en Australie

Longueur et format

  • Fixes : 10 chiffres (0 + indicatif régional + 8 chiffres).
  • Mobiles : 10 chiffres (commencent par 04).
  • Format international fixe : +61 X XXXX XXXX.
  • Format international mobile : +61 4 XX XXX XXX.

Indicatifs régionaux

  • 02 : Nouvelle‑Galles du Sud, ACT
  • 03 : Victoria, Tasmanie
  • 07 : Queensland
  • 08 : Australie‑Occidentale, Australie‑Méridionale, NT

Numéros spéciaux

  • 000 : urgences
  • 112 : urgences (mobile)
  • 13 / 1300 / 1800 : services spéciaux, numéros locaux, gratuits, etc.
