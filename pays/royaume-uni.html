Comment appeler le Royaume‑Uni depuis l’étranger

Format général

Pour appeler un numéro britannique depuis l’étranger :

Préfixe international + 44 + numéro sans le 0

Exemple :
07700 900123 → +44 7700 900123

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 44.
  • Terminez par le numéro britannique sans le 0.

Exemple depuis la France :
00 44 20 7946 0123

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +44 puis le numéro sans le 0.

Exemple :
+44 7700 900123

Structure des numéros de téléphone au Royaume‑Uni

Longueur et format

  • Les numéros britanniques font 10 chiffres (sans le 0).
  • Format national : 0XXXX XXXXXX.
  • Format international : +44 XXXX XXXXXX.
  • Le 0 est toujours retiré en international.

Préfixes géographiques (fixes)

  • 020 : Londres
  • 0161 : Manchester
  • 0141 : Glasgow
  • 029 : Cardiff
  • 028 : Irlande du Nord

Mobiles et numéros spéciaux

  • 07 : mobiles
  • 0800 / 0808 : numéros gratuits
  • 084 / 087 : numéros surtaxés
  • 09 : services premium
  • 999 / 112 : urgences

FAQ — Appeler le Royaume‑Uni avec l’indicatif +44

Pourquoi faut‑il enlever le 0 du numéro britannique ?

Le 0 est un préfixe national. Lors d’un appel international, il est remplacé par +44.

Comment reconnaître un numéro britannique ?

Les fixes commencent par 01 / 02, les mobiles par 07, et en international par +44.

Comment appeler un mobile britannique ?

Format : +44 7XXX XXXXXX.

Les numéros 09 sont‑ils surtaxés ?

Oui, ce sont des numéros premium facturés selon le service.
