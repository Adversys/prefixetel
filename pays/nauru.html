Comment appeler Nauru depuis l’étranger

Format général

Pour appeler un numéro nauruan depuis l’étranger :

Préfixe international + 674 + numéro local (7 chiffres)

Exemple :
5551234 → +674 5551234

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 674.
  • Ajoutez le numéro local à 7 chiffres.

Exemple depuis la France :
00 674 5551234

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +674 puis le numéro.

Exemple :
+674 8881234

Structure des numéros de téléphone à Nauru

Longueur et format

  • Numéros fixes : 7 chiffres.
  • Mobiles : 7 chiffres.
  • Format national : XXXXXXX.
  • Format international : +674 XXXXXXX.
  • Aucun 0 à retirer (pas de préfixe national).

Préfixes et types de numéros

  • 444XXXX : lignes fixes
  • 555XXXX : lignes fixes / entreprises
  • 888XXXX : mobiles
  • 777XXXX : services divers

Numéros spéciaux

  • 110 : police
  • 111 : urgences
  • 112 : ambulance

FAQ — Appeler Nauru avec l’indicatif +674

Nauru a‑t‑il un préfixe national à retirer ?

Non. Les numéros ne commencent jamais par 0.

Comment reconnaître un numéro de Nauru ?

Les fixes commencent par 444 ou 555, les mobiles par 888.

Le 111 fonctionne‑t‑il ?

Oui, c’est le numéro d’urgence principal du pays.
