Comment appeler la Moldavie depuis l’étranger

Format général

Pour appeler un numéro moldave depuis l’étranger :

Préfixe international + 373 + indicatif local + numéro

Exemple :
(22) 123 456 → +373 22 123 456

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 373.
  • Ajoutez l’indicatif local (ex : 22 pour Chișinău).
  • Terminez par le numéro local.

Exemple depuis la France :
00 373 22 123 456

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +373 puis l’indicatif.

Exemple :
+373 69 987 654

Structure des numéros de téléphone en Moldavie

Longueur et format

  • Numéros à 8 chiffres (hors indicatif international).
  • Format national : 0AA XXX XXX.
  • Format international : +373 AA XXX XXX.
  • Le 0 national disparaît à l’international.

Principaux indicatifs locaux

  • 22 : Chișinău
  • 231 : Bălți
  • 299 : Comrat
  • 555 : Tiraspol (Transnistrie)
  • 60 / 68 / 69 / 79 : mobiles

Numéros spéciaux

  • 112 : urgences
  • 902 : police
  • 903 : ambulance
  • 901 : pompiers

FAQ — Appeler la Moldavie avec l’indicatif +373

Pourquoi l’indicatif de la Moldavie est-il +373 ?

L’indicatif +373 a été attribué après l’indépendance du pays, remplaçant l’ancien indicatif soviétique.

Faut‑il enlever un 0 national ?

Oui. Le 0 utilisé dans le format national ne doit pas être composé depuis l’étranger.

Comment reconnaître un numéro moldave ?

Il commence par +373 suivi d’un indicatif local ou mobile.

Le 112 fonctionne-t-il en Moldavie ?

Oui, le 112 est le numéro d’urgence national.
