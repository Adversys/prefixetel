Comment appeler le Japon depuis l’étranger

Format général

Pour appeler un numéro japonais depuis l’étranger :

Préfixe international + 81 + numéro sans le 0

Exemple :
090 1234 5678 → +81 90 1234 5678

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 81.
  • Terminez par le numéro japonais sans le 0.

Exemple depuis la France :
00 81 3 1234 5678

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +81 puis le numéro sans le 0.

Exemple :
+81 90 1234 5678

Structure des numéros de téléphone au Japon

Longueur et format

  • Les numéros japonais font 10 ou 11 chiffres.
  • Format national : 0X-XXXX-XXXX ou 0XX-XXXX-XXXX.
  • Format international : +81 X-XXXX-XXXX.
  • Le 0 est toujours retiré en international.

Préfixes géographiques (fixes)

  • 03 : Tokyo
  • 06 : Osaka
  • 011 : Sapporo
  • 052 : Nagoya
  • 075 : Kyoto

Mobiles et numéros spéciaux

  • 070, 080, 090 : mobiles
  • 0120 : numéros gratuits
  • 0570 : numéros payants
  • 110 : police
  • 119 : pompiers / ambulance

FAQ — Appeler le Japon avec l’indicatif +81

Pourquoi faut‑il enlever le 0 du numéro japonais ?

Le 0 est un préfixe national. Lors d’un appel international, il est remplacé par +81.

Comment reconnaître un numéro japonais ?

Les fixes commencent par 0X, les mobiles par 070 / 080 / 090, et en international par +81.

Comment appeler un mobile japonais ?

Format : +81 70/80/90 XXXX XXXX.

Les numéros 0570 sont‑ils surtaxés ?

Oui, ce sont des numéros payants facturés selon le service.
