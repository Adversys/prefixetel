Comment appeler Kiribati depuis l’étranger

Format général

Pour appeler un numéro de Kiribati depuis l’étranger :

Préfixe international + 686 + numéro local (5 chiffres)

Exemple :
31234 → +686 31234

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 686.
  • Ajoutez le numéro local à 5 chiffres.

Exemple depuis la France :
00 686 31234

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +686 puis le numéro.

Exemple :
+686 71234

Structure des numéros de téléphone à Kiribati

Longueur et format

  • Numéros fixes : 5 chiffres.
  • Mobiles : 5 chiffres.
  • Format national : XXXXX.
  • Format international : +686 XXXXX.
  • Aucun 0 à retirer (pas de préfixe national).

Préfixes et types de numéros

  • 3XXXX : lignes fixes
  • 7XXXX : mobiles
  • 90XXX : services spéciaux

Numéros spéciaux

  • 999 : urgences
  • 994 : ambulance
  • 993 : police

FAQ — Appeler Kiribati avec l’indicatif +686

Faut‑il retirer un 0 avant d’appeler ?

Non. Les numéros de Kiribati ne commencent pas par 0.

Comment reconnaître un numéro de Kiribati ?

Les fixes commencent par 3, les mobiles par 7.

Le 999 fonctionne‑t‑il ?

Oui, c’est le numéro d’urgence principal à Kiribati.
