Comment appeler les États‑Unis depuis l’étranger

Format général

Pour appeler un numéro américain depuis l’étranger :

Préfixe international + 1 + indicatif régional + numéro

Exemple :
(415) 555‑1234 → +1 415 555 1234

Important : il n’y a jamais de 0 à retirer.

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 1.
  • Ajoutez l’indicatif régional (3 chiffres).
  • Terminez par le numéro (7 chiffres).

Exemple depuis la France :
00 1 212 555 7890

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +1 puis l’indicatif régional et le numéro.

Exemple :
+1 305 555 9876

Structure des numéros de téléphone aux États‑Unis

Longueur et format

  • Format standard : +1 AAA NXX XXXX.
  • AAA = indicatif régional (3 chiffres).
  • NXX XXXX = numéro local (7 chiffres).
  • Aucun 0 ni préfixe national.

Indicatifs régionaux populaires

  • 212 / 646 / 917 : New York
  • 213 / 310 / 424 : Los Angeles
  • 305 / 786 : Miami
  • 312 / 773 : Chicago
  • 415 / 628 : San Francisco

Numéros spéciaux

  • 800 / 888 / 877 / 866 : numéros gratuits
  • 900 : numéros surtaxés
  • 911 : urgences

FAQ — Appeler les États‑Unis avec l’indicatif +1

Pourquoi les USA utilisent‑ils l’indicatif +1 ?

Les États‑Unis font partie du plan de numérotation nord‑américain (NANP), partagé avec le Canada et plusieurs territoires.

Faut‑il enlever un 0 comme en Europe ?

Non. Les numéros américains ne commencent jamais par 0.

Comment reconnaître un numéro américain ?

Il commence par un indicatif régional à 3 chiffres, suivi d’un numéro à 7 chiffres, et par +1 en international.

Les numéros 900 sont‑ils surtaxés ?

Oui, ce sont des numéros premium facturés selon le service.
