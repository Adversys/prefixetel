Comment appeler Pitcairn depuis l’étranger

Format général

Pitcairn utilise l’indicatif néo‑zélandais +64.

Préfixe international + 64 + numéro local (6 chiffres)

Exemple :
312345 → +64 312345

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 64.
  • Ajoutez le numéro local.

Exemple depuis la France :
00 64 312345

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +64 puis le numéro.

Exemple :
+64 312345

Structure des numéros de téléphone à Pitcairn

Longueur et format

  • Numéros fixes : 6 chiffres.
  • Aucun réseau mobile.
  • Format national : XXXXXX.
  • Format international : +64 XXXXXX.
  • Aucun 0 à retirer.

Particularités

  • Très faible population (~50 habitants).
  • Infrastructure télécom minimale.
  • Appels routés via la Nouvelle‑Zélande.
  • Pas de numéros mobiles.

Numéros spéciaux

  • 111 : urgences (via NZ)
  • +64 3 12345 : administration locale

FAQ — Appeler Pitcairn avec l’indicatif +64
