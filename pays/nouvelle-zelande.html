Comment appeler la Nouvelle‑Zélande depuis l’étranger

Format général

Pour appeler un numéro néo‑zélandais depuis l’étranger :

Préfixe international + 64 + indicatif régional + numéro local

Exemple fixe (Auckland) :
(09) 123 4567 → +64 9 123 4567

Exemple mobile :
021 123 4567 → +64 21 123 4567

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00).
  • Ajoutez 64.
  • Retirez le 0 initial.

Exemple depuis la France :
00 64 9 123 4567

Depuis un mobile

  • Utilisez +64.
  • Retirez le 0 initial.

Exemple :
+64 21 123 4567

Structure des numéros en Nouvelle‑Zélande

Longueur et format

  • Fixes : 8 à 9 chiffres (0 + indicatif + numéro).
  • Mobiles : 9 à 10 chiffres (commencent par 02).
  • Format international fixe : +64 X XXX XXXX.
  • Format international mobile : +64 2X XXX XXXX.

Indicatifs régionaux

  • 03 : Sud
  • 04 : Wellington
  • 06 : Centre
  • 07 : Nord‑centre
  • 09 : Auckland

Numéros spéciaux

  • 111 : urgences
  • 0800 : numéros gratuits
  • 0900 : numéros surtaxés
