Comment appeler l’Allemagne depuis l’étranger

Format général

Pour appeler un numéro allemand depuis l’étranger :

Préfixe international + 49 + numéro sans le 0

Exemple :
0176 12345678 → +49 176 12345678

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 49.
  • Terminez par le numéro allemand sans le 0.

Exemple depuis la France :
00 49 30 123456

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +49 puis le numéro sans le 0.

Exemple :
+49 176 12345678

Structure des numéros de téléphone en Allemagne

Longueur et format

  • Les numéros allemands font entre 5 et 11 chiffres (sans le 0).
  • Format national : 0XXX XXXXXX.
  • Format international : +49 XXX XXXXXX.

Préfixes géographiques (fixes)

  • 030 : Berlin
  • 040 : Hambourg
  • 069 : Francfort
  • 089 : Munich
  • 0211 : Düsseldorf

Mobiles et numéros spéciaux

  • 015, 016, 017 : mobiles
  • 0800 : numéros gratuits
  • 0900 : numéros surtaxés
  • 110 : police
  • 112 : urgences

FAQ — Appeler l’Allemagne avec l’indicatif +49

Pourquoi faut‑il enlever le 0 du numéro allemand ?

Le 0 est un préfixe national. Lors d’un appel international, il est remplacé par +49.

Comment reconnaître un numéro allemand ?

Les numéros allemands commencent par 0 en national et par +49 en international.

Comment appeler un mobile allemand ?

Remplacez le 0 par +49 : 0176 12345678 → +49 176 12345678.

Les numéros 0900 sont‑ils surtaxés ?

Oui, ce sont des numéros premium facturés plus cher selon le service.
