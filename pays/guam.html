Comment appeler Guam depuis l’étranger

Format général

Guam utilise le plan de numérotation nord‑américain (NANP).

Préfixe international + 1 + 671 + numéro à 7 chiffres

Exemple :
671‑555‑1234 → +1 671 555 1234

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 1 671.
  • Terminez par le numéro local à 7 chiffres.

Exemple depuis la France :
00 1 671 555 1234

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +1 671 puis le numéro.

Exemple :
+1 671 777 4567

FAQ — Appeler Guam avec l’indicatif +1 671

Guam a‑t‑il son propre indicatif ?

Non. Guam utilise l’indicatif nord‑américain +1 avec le code régional 671.

Faut‑il retirer un 0 avant d’appeler ?

Non. Le format NANP ne comporte pas de 0 national.

Le 911 fonctionne‑t‑il ?

Oui, comme dans tous les territoires américains.
