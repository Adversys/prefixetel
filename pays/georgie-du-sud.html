Comment appeler la Géorgie du Sud depuis l’étranger

Format général

Le territoire utilise l’indicatif +500, partagé avec les îles Malouines.

Préfixe international + 500 + numéro local (5 chiffres)

Exemple :
12345 → +500 12345

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 500.
  • Terminez par le numéro local à 5 chiffres.

Exemple depuis la France :
00 500 12345

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +500 puis le numéro.

Exemple :
+500 23456

Structure des numéros en Géorgie du Sud

Longueur et format

  • Numéros courts à 5 chiffres.
  • Format national : XXXXX.
  • Format international : +500 XXXXX.
  • Aucun 0 à retirer.

Particularités télécom

  • Aucun réseau mobile commercial.
  • Communications assurées via les îles Malouines.
  • Usage principalement administratif et scientifique.
  • Nombre très limité de lignes fixes.
  • Nombreuses communications via satellite.

Numéros spéciaux

  • 999 : urgences (routé via le Royaume‑Uni)
  • 112 : urgences internationales

FAQ — Appeler la Géorgie du Sud avec l’indicatif +500

L’indicatif +500 est‑il correct ?

Oui. Le territoire utilise l’indicatif +500, partagé avec les îles Malouines, conformément à l’UIT.

Y a‑t‑il un réseau mobile ?

Non. Il n’existe aucun réseau mobile commercial sur le territoire.

Qui utilise ces numéros ?

Principalement les équipes scientifiques, les autorités locales et les stations de recherche.

Faut‑il retirer un 0 national ?

Non. Les numéros commencent directement par 5 chiffres.
