Comment appeler l’Italie depuis l’étranger

Format général

Pour appeler un numéro italien depuis l’étranger :

Préfixe international + 39 + numéro complet

Exemple :
347 123 4567 → +39 347 123 4567

Important : en Italie, le zéro des numéros fixes est conservé même en international.

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 39.
  • Terminez par le numéro italien (avec le 0 si fixe).

Exemple depuis la France :
00 39 06 123 4567

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +39 suivi du numéro.

Exemple :
+39 347 123 4567

Structure des numéros de téléphone en Italie

Longueur et format

  • Les numéros italiens font entre 9 et 11 chiffres.
  • Format national : 0X XXX XXXX (fixes) ou 3XX XXX XXXX (mobiles).
  • Format international : +39 X XXX XXXX.
  • Le 0 des fixes est conservé même en appel international.

Préfixes géographiques (fixes)

  • 06 : Rome
  • 02 : Milan
  • 081 : Naples
  • 041 : Venise
  • 055 : Florence

Mobiles et numéros spéciaux

  • 3XX : mobiles (TIM, Vodafone, WindTre, Iliad)
  • 800 : numéros gratuits
  • 199 : numéros payants
  • 112 : urgences

FAQ — Appeler l’Italie avec l’indicatif +39

Faut-il enlever le 0 du numéro italien ?

Non. En Italie, le 0 des numéros fixes fait partie intégrante du numéro et doit être conservé en international.

Comment reconnaître un numéro italien ?

Les fixes commencent par 0, les mobiles par 3, et les numéros internationaux par +39.

Comment appeler un mobile italien ?

Format : +39 3XX XXX XXXX.

Les numéros 199 sont-ils surtaxés ?

Oui, ce sont des numéros payants dont le coût dépend du service.
