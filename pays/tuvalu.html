Comment appeler Tuvalu depuis l’étranger

Format général

Pour appeler un numéro de Tuvalu depuis l’étranger :

Préfixe international + 688 + numéro local (5 chiffres)

Exemple :
20123 → +688 20123

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 688.
  • Ajoutez le numéro local à 5 chiffres.

Exemple depuis la France :
00 688 20123

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +688 puis le numéro.

Exemple :
+688 50123

Structure des numéros de téléphone à Tuvalu

Longueur et format

  • Numéros fixes : 5 chiffres.
  • Mobiles : 5 chiffres.
  • Format national : XXXXX.
  • Format international : +688 XXXXX.
  • Aucun 0 à retirer (pas de préfixe national).

Préfixes et types de numéros

  • 2XXXX : lignes fixes
  • 5XXXX : mobiles
  • 90XXX : services spéciaux

Numéros spéciaux

  • 911 : urgences
  • 922 : police
  • 933 : ambulance

FAQ — Appeler Tuvalu avec l’indicatif +688

Faut‑il retirer un 0 avant d’appeler ?

Non. Les numéros de Tuvalu ne commencent pas par 0.

Comment reconnaître un numéro de Tuvalu ?

Les fixes commencent par 2, les mobiles par 5.

Le 911 fonctionne‑t‑il ?

Oui, c’est le numéro d’urgence principal à Tuvalu.
