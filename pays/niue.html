Comment appeler Niue depuis l’étranger

Format général

Pour appeler un numéro de Niue depuis l’étranger :

Préfixe international + 683 + numéro local (4 chiffres)

Exemple :
4001 → +683 4001

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 683.
  • Ajoutez le numéro local à 4 chiffres.

Exemple depuis la France :
00 683 4001

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +683 puis le numéro.

Exemple :
+683 5012

FAQ — Appeler Niue avec l’indicatif +683

Faut‑il retirer un 0 avant d’appeler ?

Non. Les numéros de Niue ne commencent pas par 0.

Comment reconnaître un numéro de Niue ?

Les fixes commencent par 4, les mobiles par 5.

Le 999 fonctionne‑t‑il ?

Oui, c’est le numéro d’urgence principal du territoire.
