Comment appeler les Samoa depuis l’étranger

Format général

Pour appeler un numéro samoan depuis l’étranger :

Préfixe international + 685 + numéro local (5 à 7 chiffres)

Exemple :
721234 → +685 721234

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 685.
  • Ajoutez le numéro local.

Exemple depuis la France :
00 685 721234

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +685 puis le numéro.

Exemple :
+685 751234

Structure des numéros de téléphone aux Samoa

Longueur et format

  • Numéros fixes : 5 à 6 chiffres.
  • Mobiles : 7 chiffres.
  • Format national : XXXXX ou XXXXXXX.
  • Format international : +685 XXXXX.
  • Aucun 0 à retirer.

Préfixes et types de numéros

  • 2 / 3 : lignes fixes
  • 7 / 8 : mobiles
  • 600 : services spéciaux

Numéros spéciaux

  • 999 : urgences
  • 995 : ambulance
  • 994 : pompiers

FAQ — Appeler les Samoa avec l’indicatif +685

Faut‑il retirer un 0 avant d’appeler ?

Non. Les numéros samoans ne commencent pas par 0.

Comment reconnaître un numéro samoan ?

Les fixes commencent par 2 ou 3, les mobiles par 7 ou 8.

Le 999 fonctionne‑t‑il ?

Oui, c’est le numéro d’urgence principal aux Samoa.
