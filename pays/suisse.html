Comment appeler la Suisse depuis l’étranger

Format général

Pour appeler un numéro suisse depuis l’étranger :

Préfixe international + 41 + numéro sans le 0

Exemple :
079 123 45 67 → +41 79 123 45 67

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 41.
  • Terminez par le numéro suisse sans le 0.

Exemple depuis la France :
00 41 22 123 45 67

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +41 puis le numéro sans le 0.

Exemple :
+41 79 123 45 67

Structure des numéros de téléphone en Suisse

Longueur et format

  • Les numéros suisses font 9 chiffres (sans le 0).
  • Format national : 0XX XXX XX XX.
  • Format international : +41 XX XXX XX XX.

Préfixes géographiques (fixes)

  • 021 : Lausanne
  • 022 : Genève
  • 026 : Fribourg
  • 027 : Valais
  • 031 : Berne
  • 041 : Lucerne
  • 044 / 043 : Zurich

Mobiles et numéros spéciaux

  • 076, 077, 078, 079 : mobiles
  • 0800 : numéros gratuits
  • 084x : numéros partagés
  • 090x : numéros surtaxés

FAQ — Appeler la Suisse avec l’indicatif +41

Pourquoi faut-il enlever le 0 du numéro suisse ?

Le 0 est un préfixe national. Lors d’un appel international, il est remplacé par +41.

Comment reconnaître un numéro suisse ?

Les numéros suisses commencent par 0 en national et par +41 en international.

Comment appeler un mobile suisse ?

Remplacez le 0 par +41 : 079 123 45 67 → +41 79 123 45 67.

Les numéros 090x sont-ils surtaxés ?

Oui, ce sont des numéros premium facturés plus cher selon le service.
