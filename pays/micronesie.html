Comment appeler la Micronésie depuis l’étranger

Format général

Pour appeler un numéro micronésien depuis l’étranger :

Préfixe international + 691 + numéro local (7 chiffres)

Exemple :
3201234 → +691 3201234

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 691.
  • Ajoutez le numéro local à 7 chiffres.

Exemple depuis la France :
00 691 3201234

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +691 puis le numéro.

Exemple :
+691 9201234

Structure des numéros de téléphone en Micronésie

Longueur et format

  • Numéros fixes : 7 chiffres.
  • Mobiles : 7 chiffres.
  • Format national : XXXXXXX.
  • Format international : +691 XXXXXXX.
  • Aucun 0 à retirer (pas de préfixe national).

Préfixes et types de numéros

  • 3XXXXX : lignes fixes (Pohnpei)
  • 2XXXXX : lignes fixes (Chuuk)
  • 9XXXXX : mobiles
  • 7XXXXX : services divers

Numéros spéciaux

  • 911 : urgences
  • 411 : renseignements
  • 3201990 : police (Pohnpei)

FAQ — Appeler la Micronésie avec l’indicatif +691

Faut‑il retirer un 0 avant d’appeler ?

Non. Les numéros micronésiens ne commencent pas par 0.

Comment reconnaître un numéro de Micronésie ?

Les fixes commencent par 2 ou 3, les mobiles par 9.

Le 911 fonctionne‑t‑il ?

Oui, c’est le numéro d’urgence principal dans le pays.
