Comment appeler les Samoa américaines depuis l’étranger

Format général

Les Samoa américaines utilisent le plan de numérotation nord‑américain (NANP).

Préfixe international + 1 + 684 + numéro local (7 chiffres)

Exemple :
684‑733‑1234 → +1 684 733 1234

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 1 684.
  • Ajoutez le numéro local à 7 chiffres.

Exemple depuis la France :
00 1 684 733 1234

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +1 684 puis le numéro.

Exemple :
+1 684 258 9876

FAQ — Appeler les Samoa américaines avec l’indicatif +1 684

Les Samoa américaines ont‑elles leur propre indicatif ?

Non. Elles utilisent l’indicatif nord‑américain +1 avec le code régional 684.

Faut‑il retirer un 0 avant d’appeler ?

Non. Le format NANP ne comporte pas de préfixe national.

Le 911 fonctionne‑t‑il ?

Oui, comme dans tous les territoires américains.
