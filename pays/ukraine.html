Comment appeler l’Ukraine depuis l’étranger

Format général

Pour appeler un numéro ukrainien depuis l’étranger :

Préfixe international + 380 + indicatif local + numéro

Exemple :
(44) 234 56 78 → +380 44 234 56 78

Depuis un téléphone fixe

  • Composez le préfixe international (ex : 00 en Europe).
  • Ajoutez 380.
  • Ajoutez l’indicatif local (ex : 44 pour Kyiv).

Exemple depuis la France :
00 380 44 234 56 78

Depuis un mobile

  • Utilisez le signe +.
  • Tapez +380 puis l’indicatif.

Exemple :
+380 67 987 6543

Structure des numéros de téléphone en Ukraine

Longueur et format

  • Numéros à 9 chiffres (hors indicatif international).
  • Format national : 0AA XXX XXXX.
  • Format international : +380 AA XXX XXXX.
  • Le 0 national disparaît à l’international.

Principaux indicatifs locaux

  • 44 : Kyiv
  • 32 : Lviv
  • 48 : Odessa
  • 56 : Dnipro
  • 62 : Donetsk
  • 63 / 66 / 67 / 68 / 73 / 93 / 95 / 96 / 97 / 98 / 99 : mobiles

Numéros spéciaux

  • 112 : urgences
  • 101 : pompiers
  • 102 : police
  • 103 : ambulance

FAQ — Appeler l’Ukraine avec l’indicatif +380

Pourquoi l’indicatif de l’Ukraine est‑il +380 ?

L’indicatif +380 a été attribué après l’indépendance du pays en 1991, remplaçant l’ancien indicatif soviétique.

Faut‑il enlever un 0 national ?

Oui. Le 0 utilisé dans le format national ne doit pas être composé depuis l’étranger.

Comment reconnaître un numéro ukrainien ?

Il commence par +380 suivi d’un indicatif local ou mobile.

Le 112 fonctionne‑t‑il en Ukraine ?

Oui, le 112 est pleinement opérationnel dans tout le pays.
