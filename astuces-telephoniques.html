Astuces téléphoniques (France)

Codes utiles, blocage d’appels, masquage de numéro, filtrage, services gratuits… Toutes les astuces pour maîtriser votre ligne fixe ou mobile.

Bloquer un numéro de téléphone fixe

Vous êtes harcelé par un numéro fixe (démarchage, spam, appels répétés) ? Voici comment bloquer un numéro selon votre opérateur.

Chez Free

  • Activer le filtrage des appels entrants : *351#
  • Désactiver : #35#
  • Bloquer un numéro : *351*numéro#

Exemple pour bloquer 0171025803 :

*351*0171025803#

Autres opérateurs (Orange, SFR, Bouygues)

Sur les lignes fixes de ces opérateurs, il n’est pas possible de bloquer un numéro spécifique.

Exception SFR

Vous pouvez bloquer les appels masqués :

  • Activer : *82#
  • Désactiver : #82#

Bloquer les démarchages téléphoniques avec Bloctel

Bloctel est un service gouvernemental permettant de ne plus recevoir d’appels commerciaux. Les entreprises de prospection doivent obligatoirement retirer de leurs listes les numéros inscrits.

Inscription : bloctel.gouv.fr

Après inscription, vous recevez un numéro de compte permettant de déposer une réclamation si vous recevez encore des appels commerciaux.

Sanctions

Les entreprises qui ne respectent pas Bloctel encourent jusqu’à 2 ans de prison et 150 000 € d’amende.

Législation 2023 sur le démarchage téléphonique

Préfixes autorisés pour les démarcheurs

Depuis 2023, les démarcheurs ne peuvent plus appeler depuis un numéro en 06 ou 07.

Plages horaires autorisées

  • 10h – 13h
  • 14h – 20h

Aucun appel commercial autorisé le samedi, dimanche et jours fériés.

Signaler un SMS frauduleux

Transférez le SMS au 33700 (service anti‑spam officiel).

Masquer son numéro de téléphone

Depuis une ligne fixe

  • Free : *31* (désactiver : #31#)
  • Orange : 3651
  • Bouygues : *1651
  • SFR : 3651
  • Numéricable : *31#

Depuis un mobile

Code universel : #31#

Bloquer ou filtrer les appels sortants

Bloquer tous les appels sortants

  • Activer : *33*0000#
  • Désactiver : #33*0000#

Filtrer certains numéros sortants

  • Activer : *341*0000#
  • Ajouter un numéro : *341*0000*numéro#
  • Supprimer la liste : #341*0000#
  • Désactiver : #34*0000#

Autres services téléphoniques gratuits

  • Accès messagerie : **1
  • Dernier appelant non répondu : 3131
  • Avertissement d’appel en instance : *43# (désactiver : #43#)
  • Transfert d’appel : *21*numéro# (désactiver : #21#)
